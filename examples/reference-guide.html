What is EnergyHelix 3D?

EnergyHelix 3D is a React/Three.js visualization library for energy futures data. It renders candlestick data as a 3D helix structure where buyers and sellers spiral around a central axis, with price mapped to the Z-axis.

React Three.js r3f TypeScript

Quick Start


// 1. Install dependencies
npm install energy-helix-3d @react-three/fiber @react-three/drei three

// 2. Import and use
import { EnergyHelix } from 'energy-helix-3d';
import { Canvas } from '@react-three/fiber';

function App() {
  return (
    <Canvas>
      <EnergyHelix
        candles={candleData}
        symbol="CL"
      />
    </Canvas>
  );
}

HelixCandle Data Structure

interface HelixCandle {
  time: number;        // Unix timestamp (seconds)
  open: number;        // Opening price
  high: number;        // High price
  low: number;         // Low price
  close: number;       // Closing price
  volume: number;      // Total volume
  delta?: number;      // Optional: Buy - Sell volume
  buyVolume?: number;  // Optional: Buyer volume
  sellVolume?: number; // Optional: Seller volume
  weatherImpact?: number; // Optional: Weather impact score (0-100)
}

Events & Callbacks

Handle user interactions with the helix

HelixCallbacks Interface

export interface HelixCallbacks {
  onCandleSelect?: (index: number | null) => void;
  onCandleHover?: (index: number | null) => void;
}

onCandleSelect

Called when a user clicks on a buyer or seller node. Passes the candle index or null if deselected. 

// Usage example
<EnergyHelix
  candles={data}
  symbol="CL"
  callbacks={{
    onCandleSelect: (index) => {
      if (index !== null) {
        const candle = data[index];
        console.log('Selected:', candle.close);
      }
    }
  }}
/>

onCandleHover

Called when the cursor enters or leaves a node. Use for tooltips or highlighting external elements. 

// Usage example
const [hoveredCandle, setHoveredCandle] = useState(null);

<EnergyHelix
  candles={data}
  symbol="CL"
  callbacks={{
    onCandleHover: (index) => {
      setHoveredCandle(index);
    }
  }}
/>

Interaction Behavior

Click to Select
Click any node to select. Click again or click elsewhere to deselect.

Nodes change color and scale on hover. Cursor changes to pointer.
Selection Ring

Info Label

Complete Example

import { useState } from 'react';
import { EnergyHelix } from 'energy-helix-3d';
import { Canvas } from '@react-three/fiber';

function TradingView({ candleData }) {
  const [selected, setSelected] = useState(null);
  const [hovered, setHovered] = useState(null);

  return (
    <div className="relative">
      {/* Info panel for selected candle */}
      {selected !== null && (
        <div className="absolute top-4 left-4 bg-gray-900/90
                        backdrop-blur p-4 rounded-lg border
                        border-amber-500/30 z-10">
          <div className="text-sm text-gray-400">Selected</div>
          <div className="text-lg font-bold">
            {candleData[selected].close.toFixed(2)}
          </div>
        </div>
      )}

      <Canvas camera={{ position: [6, 4, 6], fov: 50 }}>
        <ambientLight intensity={0.4} />
        <pointLight position={[10, 10, 10]} />

        <EnergyHelix
          candles={candleData}
          symbol="CL"
          selectedIndex={selected}
          options={{
            showEIALayer: true,
            showWeatherLayer: true,
            autoRotate: true,
          }}
          callbacks={{
            onCandleSelect: setSelected,
            onCandleHover: setHovered,
          }}
        />
      </Canvas>
    </div>
  );
}

Performance Guide

        
Optimize your helix visualizations

      


      

        
        

          

            
            Best Practices
          

          

            

              
Use Instances

              
BuyerNodes and SellerNodes use Three.js Instances for efficient batch rendering of up to 500 nodes each.

            

            

              
Memoize Candle Data

              
Wrap candle arrays in useMemo() to prevent unnecessary recalculations.

              
const candles = useMemo(() => processData(raw), [raw]);

            

            

              
Limit Candle Count

              
Optimal: 100-300 candles. For more, consider windowing or aggregation.

            

            

              
Conditional Layers

              
Disable unused layers (showWeatherLayer, showVolumeProfile) to reduce draw calls.

import { useMemo } from 'react';
import { TorusGeometry, MeshBasicMaterial, Mesh } from 'three';

interface TorusRingProps {
  radius?: number;
  tube?: number;
  color?: string;
  animated?: boolean;
}

export function TorusRing({ 
  radius = 1.0, 
  tube = 0.3, 
  color = '#F59E0B',
  animated = true
}: TorusRingProps) {
  
  const geometry = useMemo(() => {
    return new TorusGeometry(radius, tube, 16, 100);
  }, [radius, tube]);

  return (
    <mesh geometry={geometry}>
      <meshBasicMaterial 
        color={color} 
        transparent
        opacity={0.8}
      />
    </mesh>
  );
}
